
1. Tento program slouží k nahrávání souboru z vašeho počítače na server

2. Aby se tento program mohl spustit musíme si naninstalovat node.js a knihovny multer a express

3. Po této intalaci se dáme pomocí terminálu do složky kde máme soubory v mém případě htdocs/xamp

4. Spustíme node js pomocí příkazu node server.js

5. Spustíme si index.html 

6. Poté by program měl běžet, vyberete si soubor, který chcete nahrát a zmáčknete tlačítko upload po zmačknutí tlačítka se soubor automaticky
